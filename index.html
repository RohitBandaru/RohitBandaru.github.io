Cornell Autonomous Bicycle Team

Cornell Autonomous Bicycle Team is an engineering project team composed of a variety of different engineering majors, with the goal of developing an efficient and intelligent self driving bicycle.

I also led development of the web application component of the system. This was meant to provide an interface to create a bike route with Google Maps and send the navigation data to the bicycle hardware. It also receives physical debugging data such as speed, position, and lean angle, and stores it in a PostgreSQL database.

The main functions of the web application are navigation and data storage and analysis. The web server sends navigation data to the bicycle using the Google Maps API. It receives data from the hardware and stores it in a PostgreSQL database. There is also an interface to download and interact with the data. This data is physical state data from the bicycle (velocity, lean angle, etc.) and is used to optimize performance.
